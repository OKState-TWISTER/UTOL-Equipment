========================================================================
    DYNAMIC LINK LIBRARY : IPconnections Project Overview
========================================================================

DLL providing 2 functions used in LabVIEW to get the number of network
interfaces and to get the IP addresses and the corresponding broadcast
addresses for these interfaces.

You will find a detailed documentation of the code in Doxygen style in
the header file(s).

This project has been originally built with
Microsoft Visual Studio 2005
This version (VS2005) works with Win2000 - Win7
Visual Studio 2010 is known not to work below WinXP SP2
